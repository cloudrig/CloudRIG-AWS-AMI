# CloudRIG AWS AMI Factory

This repository contains the tool used to generated the CloudRIG AWS AMIs. 
They come with:
  * Parsec
  * NVIDIA drivers
  * Razer Surround
  * DirectX June 2010 Redist
  * Google Chrome
  * TightVNC
  * 7Zip
  * ViGEmBus (for controllers support)
  * .Net 3.5
  * Direct Play
